## 📓 Problem Description

- **문제 요약**: 서쪽의 N개의 사이트와 동쪽의 M개의 사이트를 연결할 때, 다리가 서로 겹치지 않게 연결하는 경우의 수 구하기.
    
- **입력**:
```
t (테스트 케이스 개수)
n m (서쪽 사이트 개수, 동쪽 사이트 개수)
```
    
- **출력**:
```
ans
```
    
- **제약 조건**: $0 < N \le M < 30$
